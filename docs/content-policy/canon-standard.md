# The Living Library — Canon Standard

## Purpose

The Canon Standard defines how content is classified, evaluated, and presented within The Living Library. 
It exists to maintain intellectual integrity and prevent the platform from becoming a pseudo-historical 
content farm or conspiracy platform.

## Source Type Classification

Every piece of content must be classified into one of the following categories:

| Type | Code | Description | Examples |
|------|------|-------------|---------|
| Primary Source | `PRIMARY_SOURCE` | Original historical text | Tao Te Ching, The Republic, Upanishads |
| Classical Commentary | `CLASSICAL_COMMENTARY` | Traditional commentary by recognized authorities | Proclus on Plato, Shankara on Upanishads |
| Modern Scholarship | `MODERN_SCHOLARSHIP` | Academic interpretation and analysis | Scholarly books, peer-reviewed articles |
| Comparative Analysis | `COMPARATIVE_ANALYSIS` | Cross-cultural scholarly comparison | Joseph Campbell's comparative mythology |
| Speculative | `SPECULATIVE` | Clearly marked interpretive speculation | New interpretations, contested theories |
| Community Commentary | `COMMUNITY_COMMENTARY` | User-generated discussion and commentary | Forum posts, member journals |

## Difficulty Classification

| Level | Code | Description |
|-------|------|-------------|
| Introductory | `SEED` | No prerequisites required. Accessible to general readers. |
| Foundation | `FOUNDATION` | Basic familiarity with the tradition recommended. |
| Intermediate | `STRUCTURAL` | Requires conceptual framework and background reading. |
| Advanced | `ESOTERIC` | Symbolic complexity; requires prerequisite traditions. |
| Scholarly | `SCHOLARLY` | Advanced critical study; academic background beneficial. |
| Specialist | `SPECIALIST` | Requires deep prerequisite knowledge of specific traditions. |

## Knowledge Confidence System

For AI-generated content and interpretive statements, all claims are tagged with:

| Level | Code | Meaning |
|-------|------|---------|
| Historical Consensus | `HISTORICAL_CONSENSUS` | Strong scholarly evidence; widely accepted |
| Scholarly Debated | `SCHOLARLY_DEBATED` | Active academic disagreement exists |
| Traditional Interpretation | `TRADITIONAL_INTERPRETATION` | Based on the tradition's own account |
| Speculative | `SPECULATIVE` | Weak or no direct evidence |
| Mythological/Symbolic | `MYTHOLOGICAL_SYMBOLIC` | Non-literal, symbolic domain |

## Prohibited Content

The following content types are **never** permitted in the archive:

- Fabricated ancient texts or invented "lost teachings"
- AI-generated content presented as historical fact
- Conspiracy theories or pseudo-historical claims
- Content that misrepresents, appropriates, or demeans cultural traditions
- Extremist ideology dressed in spiritual language
- Cult-promoting content or personality worship
- Anonymous texts of unknown provenance presented as authoritative

## Text Lineage Requirement

For primary source texts, the following lineage information should be provided where available:

- Manuscript tradition (oldest known manuscripts)
- Translation history
- Major translators and their approaches
- Textual variants
- Commentary traditions
- Influence tree (what later works it influenced)

## Review Process

1. **Submission**: Content submitted via admin interface
2. **Classification**: Curator assigns source type, difficulty, and confidence level
3. **Citation check**: At least one verifiable source must be attached
4. **Historical context review**: Context must be accurate and nuanced
5. **Publication**: Admin approves and publishes

## Anti-Fabrication Policy

The Living Library has zero tolerance for:
- Inventing quotes attributed to historical figures
- Presenting modern interpretations as ancient teachings
- Claiming "hidden" or "suppressed" knowledge without documentation
- Generating false scholarly consensus

When uncertainty exists, it must be stated explicitly.
